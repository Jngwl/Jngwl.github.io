DOMпјҲDocument Object Modelпјүж–ҮжЎЈеҜ№иұЎзұ»еһӢпјҢе…¶еҸҜд»Ҙе°Ҷж Үи®°еһӢж–ҮжЎЈдёӯзҡ„жүҖжңүеҶ…е®№пјҲж ҮзӯҫгҖҒж–Үжң¬гҖҒеұһжҖ§зӯүпјүйғҪе°ҒиЈ…жҲҗеҜ№иұЎ

  • дёҖдёӘHTMLж–ҮжЎЈеҠ иҪҪеҲ°еҶ…еӯҳдёӯе°ұдјҡеҪўжҲҗдёҖдёӘDOMеҜ№иұЎ

рҹ”Һ иҺ·еҸ–е…ғзҙ еҜ№иұЎ

  • getElementByID()пјҡйҖҡиҝҮе…ғзҙ IDиҺ·еҸ–еҜ№еә”зҡ„е…ғзҙ еҜ№иұЎ

  • getElementsByName()пјҡйҖҡиҝҮе…ғзҙ зҡ„nameеұһжҖ§иҺ·еҸ–з¬ҰеҗҲзҡ„жүҖжңүе…ғзҙ 

  • getElementsTagName()пјҡйҖҡиҝҮе…ғзҙ зҡ„е…ғзҙ еҗҚеұһжҖ§иҺ·еҸ–з¬ҰеҗҲзҡ„жүҖжңүе…ғзҙ пјҢеҰӮli
  • getElementsByClassName()пјҡйҖҡиҝҮе…ғзҙ зҡ„classеұһжҖ§иҺ·еҸ–з¬ҰеҗҲиҰҒжұӮзҡ„жүҖжңүе…ғзҙ пјҢеҰӮh1

рҹ”Һ е…ғзҙ еҜ№иұЎеёёи§ҒеұһжҖ§

  • valueпјҡе…ғзҙ еҜ№иұЎ.value
  • classNameпјҡе…ғзҙ еҜ№иұЎ.className
  • checkedпјҡе…ғзҙ еҜ№иұЎ.checked
    • HTMLдёӯchecked=вҖңcheckedвҖқпјҢJavaScriptдёӯиҝ”еӣһtrue
    • йҖҡиҝҮе…ғзҙ еҜ№иұЎ.checked = trueеҸҜд»Ҙдҝ®ж”№еҚ•йҖү/еӨҡйҖүжЎҶзҡ„й»ҳи®ӨйҖүдёӯзҠ¶жҖҒ
  • innerHTMLпјҡе…ғзҙ еҜ№иұЎ.innerHTMLиҝ”еӣһеҜ№иұЎзҡ„еҶ…е®№дҪ“гҖӮ

пјҲ5пјүJavaScriptдәӢд»¶

рҹӨ§ з®Җиҝ°

  • йј ж ҮжҲ–иҖ…й”®зӣҳзҡ„еҠЁдҪңз§°дёәдәӢд»¶пјҲEventпјүгҖӮеҰӮзӮ№еҮ»гҖҒжҸҗдәӨиЎЁеҚ•гҖҒеҖјеҸ‘з”ҹж”№еҸҳгҖҒйј ж Үз§»еҠЁзӯү

  • JSдәӢд»¶й©ұеҠЁжңәеҲ¶

    • дәӢд»¶жәҗпјҡжҢүй’®
    • дәӢд»¶пјҡзӮ№еҮ»дәӢд»¶ onclick
    • зӣ‘еҗ¬еҷЁпјҡж–№жі•пјҢеҰӮrun1()
    • жіЁеҶҢзӣ‘еҗ¬еҷЁпјҡ onclick=вҖңrun1()вҖқ

рҹӨ§ еёёи§ҒJavaScriptдәӢд»¶

  • зӮ№еҮ»дәӢд»¶onclick

    <body>
	<input type="button" value="зӮ№жҲ‘еј№еҮәеҜ№иҜқжЎҶ" onclick="run1()"/>
    
    <script>
        function run1(){
        	alert("еј№еҮә")
	    }
	</script>
</body>

  • з„ҰзӮ№дәӢд»¶

    • иҺ·еҸ–з„ҰзӮ№дәӢд»¶onfocusпјҡonfocus = вҖңrun1()вҖқ

  • еӨұеҺ»з„ҰзӮ№дәӢд»¶onblurпјҡonblur = вҖңrun1()вҖқ

  • еҹҹеҶ…е®№ж”№еҸҳдәӢд»¶пјҲе…ғзҙ еҖјж”№еҸҳж—¶е°ұдјҡи§ҰеҸ‘пјүпјҡ

  • еҠ иҪҪе®ҢжҜ•дәӢд»¶пјҡonload = вҖңж–№жі•еҗҚ()вҖқ

  • иЎЁеҚ•жҸҗдәӨдәӢд»¶пјҡonsubmit = вҖңreturn ж–№жі•еҗҚвҖқ

    • и°ғз”Ёзҡ„е…·дҪ“ж–№жі•дёӯпјҢиҰҒиҝ”еӣһtrueжҲ–иҖ…falseгҖӮжҖ»е…ұдёӨдёӘreturn
    • иҜҘдәӢд»¶йңҖиҰҒиҝ”еӣһbooleanзұ»еһӢзҡ„еҖјжқҘжү§иЎҢжҸҗдәӨ/йҳ»жӯўиЎЁеҚ•ж•°жҚ®зҡ„ж“ҚдҪң

  • й”®дҪҚеј№иө·дәӢд»¶пјҡonkeyup = "ж–№жі•еҗҚ()"

  • еёёз”Ёйј ж ҮдәӢд»¶

    • йј ж Үз§»е…ҘдәӢд»¶(з§»е…Ҙж—¶пјҢеј№жЎҶ)пјҡonmouseover = вҖңж–№жі•еҗҚ()вҖқ
    • йј ж Үз§»еҮәдәӢд»¶(з§»еҮәж—¶пјҢеј№жЎҶ)пјҡonmouseout = вҖңж–№жі•еҗҚ()вҖқ
    • жҜ”еҰӮзғҰдәәзҡ„QQжү«жҸҸзҷ»йҷҶзҡ„дәҢз»ҙз Ғ

рҹӨ§ JavaScriptдәӢд»¶зҡ„з»‘е®ҡж–№ејҸ

  • е…ғзҙ дәӢд»¶еҸҘжҹ„з»‘е®ҡ

    • е°ҶдәӢд»¶д»Ҙе…ғзҙ еұһжҖ§зҡ„ж–№ејҸеҶҷеҲ°ж ҮзӯҫеҶ…йғЁпјҢиҝӣиҖҢз»‘е®ҡеҜ№еә”еҮҪж•°
    • дјҳзјәзӮ№пјҡ
      • ејҖеҸ‘дҫҝжҚ·гҖҒдј еҸӮж–№дҫҝгҖҒеҸҜд»Ҙз»‘е®ҡеӨҡдёӘеҮҪж•°
      • JSе’ҢScriptд»Јз Ғй«ҳеәҰзі…еҗҲеңЁдёҖиө·пјҢдёҚеҲ©дәҺеӨҡйғЁй—Ёзҡ„йЎ№зӣ®ејҖеҸ‘з»ҙжҠӨгҖӮ
    
	<script>
		function run1(){
        	alert("run1");
        function run2(str){
            alert(str);
        }
        function run3(obj){
            alert(obj.value);
        }
    }
	</script>

        <!---- дёәдәӢд»¶з»‘е®ҡдёҖдёӘжңүеҸӮеҮҪж•°пјҢжӯӨеӨ„еҸӮж•°жҳҜдёӘеӯ—з¬ҰдёІ -->

  • DOMз»‘е®ҡ

    • еҜ№иұЎ.дәӢд»¶еұһжҖ§дёҖж¬ЎеҸӘиғҪз»‘е®ҡдёҖдёӘеҮҪж•°пјҢеҗҢж—¶д№ҹж— жі•дј йҖ’еҸӮж•°
    • еҢҝеҗҚеҮҪж•°еҸҜд»Ҙз»‘е®ҡеӨҡдёӘеҮҪж•°пјҢеҗҢж—¶д№ҹеҸҜд»Ҙдј йҖ’еҸӮж•°гҖӮ

        run1();

    }
</script>
