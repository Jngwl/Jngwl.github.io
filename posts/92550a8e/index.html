本文最后更新于：2021年1月8日 晚上

一、常用Windows批处理命令

1）清除制定目录下的空文件夹

@echo off
for /f "tokens=*" %%a in ('dir /b /ad /s C:\test^|sort /r') do rd "%%a" /q 2>nul

2）批量获取指定目录下的文件名

dir E:\test /b > E:\test\asd.txt

二、kindle自定义封面

1）获取中亚官方Kindle电子书高清封面图

  • 获取电子书的ASIN编号

    • 在亚马逊官网搜索对应电子书，比如《海伯利安四部曲》
    • 在电子书基本信息界面得到对应的ASIN编号

  • 将下列网址中*替换为ASIN编号即可

    http://z2-ec2.images-amazon.com/images/P/*****.01.MAIN._SCRM_.jpg
http://s3.cn-north-1.amazonaws.com.cn/sitbweb-cn/content/*****/images/cover.jpg

2）调整封面尺寸

  • 1878×1280

3）将封面图载入对应电子书

  • 安装电子书管理软件Calibre

  • 打开对应电子书（mobi或azw3格式），选中，编辑元数据/更换封面/浏览，找到之前下载好的图片，确定即可

  • 发送到设备，搞定！

最后，强烈推荐漫画网站https://vol.moe/，kindle的第二春啊

三、关闭git中的LF will be replaced by CRLF警告

眼不见心不烦，通过将autocrlf设置为false便可关闭警告

git config --get core.autocrlf

git config core.autocrlf false

git config --get core.autocrlf

  • 若遇到fatal: not in a git directory,可先进行git init 再git config core.autocrlf false

  • Git中的“LF will be replaced by CRLF”警告详解

四、压缩清除Win10休眠文件

管理员权限打开cmd，输入，powercfg hibernate size 40

  • 其中，40表示压缩率

五、去除新版Edge的『禁用扩展』提示信息

  • 下载神奇的软件：https://lanzous.com/ibzz8la
  • 在edge的安装路径，如C:\Program Files (x86)\Microsoft\Edge\Application\81.0.416.64，将msedge.dll修改为chrome.dll
  • 以管理员权限打开所下载的软件，点击应用即可

六、过鼠标手势自动上传hexo更新

鼠标手势软件：WGestures

设置/手势/添加/执行操作--命令行，输入E: && cd OneDrive - vip.jiangnan.edu.cn\Blog && hexo d -g 即可。

七、通过自定义输入法快捷键替代Hexo n等繁琐步骤

1） 前言

之前每次写博客，都得hexo n后到指定目录下打开md文件，过于繁琐，实在头大。

开始根据简书老哥的建议，尝试过hexo n后直接用Typora打开新建的md文件，但并未成功。

之后便一直将hexo所需的片头(title、tag等格式)存放在Sticky Note中，通过Typora新建md文件后粘贴过来即可，总体步骤也有点繁琐

时至今日，突发奇想，我能不能通过快捷键直接输入这些片头信息，经百度，终寻得解决方案

2）基本流程

这里采用的是搜狗输入法，属性设置/高级/候选扩展/自定义短语设置/直接编辑配置文件，在文件末尾写入下列内容

n,1=
---
title: 
date: 2019-03-19 16:23:00
tags: []
categories: 
---

more,1=<!--more-->

  • n即代替title那一串，第一顺位展示

  • more即代替<!--more-->，第一顺位展示
